# Voltera Mobile App UI

Voltera Mobile App is a Flutter-based user interface application designed for the Voltera Electric Vehicle Charging Station Management System.

The application enables EV owners to:

* Browse and search charging stations
* View station details and availability
* Reserve charging slots
* Start and monitor charging sessions
* Manage user profiles and vehicles
* Track charging history and payments
* Receive real-time notifications
* Locate nearby charging stations using maps

This repository focuses on the mobile frontend implementation, following modern Flutter architecture and state management practices for scalability and maintainability.
